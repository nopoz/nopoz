I build tools for self-hosters and homelab operators, mostly around
Docker, Portainer, DNS, and network privacy.

### Projects

- **[hosaka](https://github.com/nopoz/hosaka)**
  Docker image update monitor with semver-aware classification, one-click
  upgrades, live console output, and notifications via Slack, Discord,
  Telegram, and more.

- **[pfsense-dnscrypt-proxy](https://github.com/nopoz/pfsense-dnscrypt-proxy)**
  A pfSense package bringing DNSCrypt, DoH, Oblivious DoH, and Anonymized
  DNS to your firewall with a full GUI. Signature-verified builds with
  SLSA provenance.

- **[portrieve](https://github.com/nopoz/portrieve)**
  Back up, restore, and migrate Portainer stacks as plain, git-friendly
  Docker Compose files. Fills the gap between blunt database backups and
  manual exports.
